# 🎨 Creative Studio - Diseño Digital Innovador

Una página web moderna, creativa y completamente responsive que showcasea servicios de diseño y desarrollo web con un diseño futurista y efectos visuales atractivos.

## ✨ Características

- **Diseño Moderno**: Gradientes vibrantes, animaciones suaves y efectos visuales cautivadores
- **Totalmente Responsive**: Funciona perfectamente en desktop, tablet y móviles
- **Progressive Web App (PWA)**: Funciona offline con Service Worker integrado
- **Rendimiento Optimizado**: Carga rápida y optimizada
- **Interactividad**: Efectos hover, animaciones de scroll y controles interactivos
- **Accesibilidad**: Código semántico y navegación accesible
- **Atajos de Teclado**: Ctrl+S, Ctrl+P, Ctrl+C para navegar rápidamente

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica moderna
- **CSS3**: Gradientes, animaciones, Flexbox y Grid
- **JavaScript (Vanilla)**: Interactividad sin dependencias
- **Service Worker**: Soporte offline (PWA)
- **Web Manifest**: Instalable como aplicación

## 📱 Características Responsivas

- Breakpoints en 768px y 480px
- Menú hamburguesa en móviles
- Layouts adaptables
- Fuentes escalables

## ⌨️ Atajos de Teclado

- **Ctrl + S**: Ir a Servicios
- **Ctrl + P**: Ir a Portafolio
- **Ctrl + C**: Ir a Contacto
- **Escape**: Cerrar menú móvil

## 🎯 Uso

1. Abre `index.html` en tu navegador
2. Navega por las diferentes secciones
3. Prueba los efectos y animaciones
4. Llena el formulario de contacto
5. Instala como app (PWA) en móviles
